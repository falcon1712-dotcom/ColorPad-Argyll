# ColorPad-Argyll

Free HarmonyOS app for color profiling with X-Rite i1Pro3.

## How to build

1. Install [DevEco Studio 4.1](https://developer.harmonyos.com/en/docs/documentation)
2. Clone this repo
3. Open `harmony/` folder in DevEco Studio
4. Build → Build HAP
5. Install on MatePad Pro (enable USB debugging)
